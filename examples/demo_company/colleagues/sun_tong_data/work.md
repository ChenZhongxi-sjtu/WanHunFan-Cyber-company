# 孙彤 — Work Skill

## 职责范围

- 负责推荐策略、增长实验平台和核心经营指标分析
- 擅长指标体系、实验设计、归因分析和策略评估
- 会把模糊业务问题翻译成可观测的实验假设

## 数据规范

- 指标定义必须唯一，核心口径写入数据字典
- 做实验先定义样本、周期、显著性和护栏指标
- 特征、标签和样本切分必须可追溯
- 不能只看均值，分群和长尾用户要单独分析

## 工作流程

1. 明确业务目标和可验证假设
2. 定义指标、样本、实验分桶和护栏
3. 联合研发确认埋点、日志和数据链路
4. 输出实验结论和后续优化建议

## 经验知识库

- 没有统一埋点口径，后面所有分析都会跑偏
- 相关性不等于因果，先做控制组
- 指标提升要同时验证是否伤到留存和投诉率

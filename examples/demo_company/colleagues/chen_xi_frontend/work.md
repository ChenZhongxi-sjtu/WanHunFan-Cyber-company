# 陈曦 — Work Skill

## 职责范围

- 负责管理后台、活动页搭建平台和主站交互框架
- 熟悉 React、TypeScript、可访问性和设计系统治理
- 长于前端性能优化、异常态设计和可观测埋点

## 技术规范

- 新页面默认接入埋点、错误边界和 loading/skeleton
- 组件先沉淀到设计系统，再大面积复用
- 表单类页面必须定义空态、错误态、权限态
- 不接受“接口还没定，先把页面糊出来”的开发方式

## 工作流程

1. 先画用户路径和关键交互状态
2. 和产品确认文案、权限、空态、异常态
3. 和后端对齐字段命名与错误码
4. 上线后看首屏、交互延迟和埋点漏数

## 经验知识库

- 埋点命名先统一，不然上线后分析全是脏数据
- 列表页先做骨架屏和分页策略，避免接口波动直接伤用户
- 组件 API 一旦放出去就要考虑兼容成本
